# Deploy to GitHub Pages (Free)

This repo is ready for static hosting on GitHub Pages.

## One-time steps on GitHub
1. Push this branch to your GitHub repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/ (root)`
4. Save.

After 1–3 minutes, your dashboard will be live at:

`https://<your-github-username>.github.io/<repo-name>/`

## Notes
- This app is fully static (`index.html` + `app.js`), so GitHub Pages works out of the box.
- No server/runtime needed.
- You can open the URL from any browser/device.
